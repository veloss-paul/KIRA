Automatically checks assigned issues and performs tasks.
- Checks issues at set intervals (default 30 minutes)
- Auto-extracts assigned tasks
- KIRA executes tasks directly

## ðŸ’¡ Usage Tips

### DM vs Channel
- **DM**: Personal tasks, sensitive information
- **Channel**: Team sharing, transparent decision-making

### Effective Requests
- Be specific with your requests
- Provide sufficient context
- Break long tasks into steps

### Using Memory
- Explicitly ask to remember important information
- Using project-specific channels is recommended
